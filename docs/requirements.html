Requirements ::.::.:..

CMScout does not require you to know PHP or even HTML (Although knowing HTML will enable you to make your own template for your site).

What CMScout however does require is:

  • A Webserver running apache 1.3.33 or newer
  • PHP 4.1.0 or newer (PHP 4.4.4/5.2.0 or higher recommended)
  • MySQL 4 or newer
  • The GD image library (At least version 2.01)
  • At least 15Mb of web space (CMScout uses about 8mb, and then you need space for photos, articles, content, etc.)
  • A basic knowledge of how to add a MySQL database and set the permisions (Most hosters have a simple control panel to help with this task, if you are unsure contact your hosting provider)
